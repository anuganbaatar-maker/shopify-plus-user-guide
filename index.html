01

Shopify Plus-д Бүртгүүлэх

Shopify Plus бол enterprise түвшний платформ — бүртгэл нь стандард Shopify-аас ялгаатай.

1

Shopify Plus-тэй холбогдох

Shopify Plus нь шууд нэвтрэн бүртгүүлэх боломжгүй — та Shopify-ийн борлуулалтын багтай холбогдох шаардлагатай.

  • shopify.com/plus руу очин "Get started" дарна
  • Одоогийн орлого, бизнесийн мэдээллээ бөглөнө (сард $150K+ орлоготой байх шаардлагатай)
  • Shopify-ийн Plus мэргэжилтэн тантай 1-3 ажлын өдрийн дотор холбогдоно
  • Demo үзүүлэлт хийж, үнийн санал авна ($2,000–$40,000/сар хооронд)
2

Гэрээ байгуулж төлбөр тохиролцох

Shopify Plus-ийн үнэ нь бизнесийн хэмжээнээс хамаарна:

Төлбөрийн хувилбар Хэмжээ Тайлбар
Стандарт $2,000/сар Сарын GMV $800K хүртэл
Хувьсах GMV-ийн 0.25% $800K-аас дээш үед автоматаар
Transaction fee 0.15–0.25% Гуравдагч талын payment ашиглавал
3

Plus Organization Admin руу нэвтрэх

Гэрээ байгуулсны дараа танд Organization Admin эрх нээгдэнэ.

  • admin.shopify.com руу нэвтэрч Organizations-аа сонгоно
  • Нэг Organization дор хүртэл 9 дэлгүүр удирдах боломжтой (Expansion Store)
  • Нэмэлт дэлгүүр нэмэхэд $250/сар нэмдэг
💡 Зөвлөмж

Монгол/Азийн зах зээлд ажиллаж байгаа бол Singapore эсвэл Japan дата сервер сонгохыг зөвлөж байна — тооцоо хийх хурд мэдэгдэхүйц нэмэгдэнэ.

4

Хэрэглэгч & Хандалтын эрх тохируулах

  • Settings → Users → Invite users гэсэн замаар хэрэглэгч нэмнэ
  • Role-based access control (RBAC): Owner, Admin, Staff, Collaborator
  • Collaborator эрхийг агентлаг/хөгжүүлэгчид өгч болно — өртөг нэмдэггүй
  • Two-factor authentication (2FA) бүх хэрэглэгчид заавал идэвхжүүлнэ
02

Дэлгүүрийн Үндсэн Тохиргоо

Online дэлгүүрийг нэг удаа зөв тохируулах нь хожим гарах маш олон асуудлыг шийднэ.

Ерөнхий тохиргоо

A

Domain тохиргоо

  • Settings → Domains → Add domain
  • DNS A record: 23.227.38.32 зааж өгнө
  • SSL автоматаар идэвхждэг (24 цаг хүлээнэ)
B

Мөнгөн тэмдэгт & татвар

  • Settings → General → Store currency (MNT сонгох боломжтой)
  • Markets → добавить Mongolia гэж тусгай зах зээл нэмнэ
  • Монголд нэмэгдсэн өртгийн албан татвар (НӨАТ) 10% тохируулна
C

Payment тохиргоо

  • Shopify Payments (зарим оронд): Settings → Payments
  • Stripe, PayPal, Monpay гэх мэт гуравдагч талын provider нэмэх
  • Test mode-оор туршиж үзнэ, дараа Live болгоно

Theme & Дизайн

🎨

Theme сонгох

Online Store → Themes. Shopify Theme Store-оос сонгох эсвэл custom theme хийлгэх. Dawn (үнэгүй), Impulse, Prestige гэсэн сонголтууд алдартай.

⚡

Metaobjects & Custom data

Plus-ийн давуу тал: Metaobjects ашиглан бараа, блог, хуудасны custom field нэмэх. Content → Metaobjects.

📦

Бүтээгдэхүүн нэмэх

Products → Add product. Bulk import хийхэд CSV файл ашиглана. Variant, inventory, shipping тохируулна.

Shopify Plus-ийн онцлог функцүүд
🤖

Shopify Flow

Захиалга баталгаажих, fraud илрүүлэх, тагнах зэрэг бизнес процессыг автоматжуулна. Код бичихгүйгээр ажиллана.

🛒

Checkout Extensibility

Plus гэрээт хэрэглэгчид checkout хуудсыг checkout.liquid-ийн оронд Checkout Extensions ашиглан customize хийх боломжтой.

🔄

B2B & Wholesale

Companies, price lists, net payment terms — B2B болон wholesale дэлгүүр Plus-ийн native feature ашиглан тохируулна.

🌐

Expansion Stores

Олон улс, олон брэнд эсвэл олон валютад тус тусдаа дэлгүүр нэмэх. Organization dashboard-аас бүгдийг нэг дороос удирдана.

Facebook & Instagram холбох алхмууд
1

Meta Business Suite бүртгэл байгуулах

business.facebook.com руу очин Business Manager бүртгэлээ шалгах эсвэл шинэ бүртгэл үүсгэнэ.

  • Facebook Page (брэндийн хуудас) нэмнэ
  • Instagram Business account холбоно
  • Ad Account үүсгэж, хил орон, валют тохируулна
  • Payment method нэмнэ (карт эсвэл prepaid)
Анхаар

Business Manager болон Personal Account нэг Meta бүртгэлтэй байх ёстой. Admin эрх олгохдоо personal email биш Business email ашиглахыг зөвлөж байна.

2

Shopify-д Facebook & Instagram App суулгах

  • Shopify Admin → Apps → App Store
  • "Facebook & Instagram" app хайж олоод Install хийнэ
  • Meta Business account-тайгаа холбоно — Pixel ID автоматаар үүснэ
  • Product catalog sync идэвхжүүлнэ (бараанууд Facebook Catalog Manager-т харагдана)
Шалгах зүйлс
✓ Facebook Page → Connected
✓ Instagram Account → Connected
✓ Ad Account → Selected
✓ Meta Pixel → Created / Connected
✓ Catalog → Syncing
✓ Data Sharing → Maximum (CAPI идэвхтэй)
3

Meta Pixel болон Conversions API тохиргоо

Pixel нь browser-side tracking. CAPI нь server-side. Хоёулаа хамт ажиллавал илүү дэлгэрэнгүй мэдээлэл цуглуулна.

Event Автоматаар Тайлбар
PageView Тийм Хуудас үзсэн
ViewContent Тийм Бараа харсан
AddToCart Тийм Сагсанд нэмсэн
InitiateCheckout Тийм Checkout эхлүүлсэн
Purchase Тийм Худалдан авалт хийсэн
Lead Гараар Contact form дүүргэсэн
CompleteRegistration Гараар Хэрэглэгч бүртгүүлсэн
4

Facebook Shop & Instagram Shopping тохиргоо

  • Meta Commerce Manager → Shops → нэмэх
  • Shopify catalog-тай холбоно → бараанууд автоматаар sync болно
  • Instagram → Tag products идэвхжүүлнэ (post, story, reel дээр tag хийх боломжтой болно)
  • Facebook Shop → Customize: collection нэмэх, дизайн тохируулах
⚠ Анхааруулга

Монгол руу хүргэлттэй бараа Meta Commerce Policy-г дагасан байх шаардлагатай. Зарим бараа (хутга, архи, эм) restricted content-д ордог тул policy шалгаарай.

5

Meta Pixel шалгах — Facebook Pixel Helper

  • Chrome → Facebook Pixel Helper extension суулгана
  • Дэлгүүрийн хуудсуудаар орж event ажиллаж байгааг шалгана
  • Meta Events Manager → Test Events дотор live event харна
  • Event Match Quality (EMQ) оноо 6+ байхыг зорино

UTM параметр ашиглах

  utm_source=facebook
  &utm_medium=paid_social
